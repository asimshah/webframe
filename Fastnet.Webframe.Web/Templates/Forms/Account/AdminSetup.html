Administrator Set up

There are no members defined and the most probable reason for this is that this is a new site. At minimum there needs to be one member registered as the Administrator - no features are available until this member is set up.

Use this form to create this member - You need to provide an email address and a password. Once the Administrator is created, you will be redirected to the Home page.

No email is sent to this email address at this stage but it will be used by the system to communicate with the Administrator. No user will be allowed to register on this site using this email address.

Note that once created, the only way to change the Administrator password is to login as the Administrator using this email and password - please make sure the credentials you create here are kept securely.
